Configuracion grafana para que salgan los 3 nodos y sus conexiones

1a query

label_replace(
  count(conexiones_activas) by (desde_nodo),
  "id", "$1", "desde_nodo", "([^:]+).*"
)

2a query

label_replace(
     label_replace(
       conexiones_activas,
       "source", "$1", "desde_nodo", "([^:]+).*"
     ),
     "target", "$1", "hacia_nodo", "([^:]+).*"
   )

Las dos query con format de Table y con type Instant, ademas la primera tiene que tener legnd {{id}}

Transformations

- lables to fields
- merge series/tables
- organize fields by name (solo dejar visible id, source, target, valueA, valueB)

Configuracion grafana para que salgan los 10 nodos y sus conexiones

1a query

label_replace(
  estado_nodo,
  "id", "$1", "nodo", "(node[0-9]+)"
)

Format: Table
Type: Instant
Legend: {{nodo}}

2a query

label_replace(
  label_replace(
    conexiones_activas,
    "source", "$1", "desde_nodo", "(node[0-9]+)"
  ),
  "target", "$1", "hacia_nodo", "(node[0-9]+)"
)

Format: Table
Type: Instant

3a query

label_replace(
  label_replace(
    latencia_conexion_ms,
    "source", "$1", "source", "(node[0-9]+)"
  ),
  "target", "$1", "target", "(node[0-9]+)"
)

Format: Table
Type: Range

Transformations

- labels to fields
- merge series/tables
- organize fields by name (solo visible id, source, target, valueC)
